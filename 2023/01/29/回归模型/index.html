多元回归分析

目的：通过研究自变量X和因变量Y的相关关系，尝试去解释Y的形成机制，进而通过X去预测Y

   步骤：

  1. 识别重要变量
    哪些X变量是同Y真的相关，哪些不是（逐步回归法）
  2. 判断相关性的方向（正相关还是负相关）
  3. 估计权重（回归系数）
    赋予不同X不同的权重（回归系数），进而得知不同变量之间的相对重要性

  回归分析的分类（依据因变量Y的类型）

   数据的分类

  • 横截面数据：在某一时间点收集的不同对象的数据，易出现异方差的问题
  • 时间序列数据：对同一对象在不同时间连续观察所取得的数据，易出现自相关的问题
  • 面板数据：横截面数据与时间序列数据综合起来的一种数据资源（多个对象，一段时间）

  不同数据类型的处理方法

  数据的收集（暂鸽~）

多元线性回归

注:线性假设不要求初始模型都成严格线性关系，自变量与因变量可通过变量替换转换成线性模型    扰动项是球型扰动项

内生性

  • 外生性：误差项（扰动项）μ与所有的自变量（解释变量）x均不相关，称该回归模型具有外生性
  • 内生性：误差项（扰动项）μ包含了未添加到回归模型中但与y有关的变量
    导致回归系数估计不准确（不满足无偏和一致性）

  解决方法-确定核心解释变量和控制变量（伪外生）

  • 核心解释变量：最需要的变量，得到对其系数的一致估计（当样本容量无限增大时，收敛于待估计参数的真值 ）
  • 控制变量：不太需要这些变量，把它们放入回归方程主要是为了 “控制住” 那些对被解释变量（y）有影响的遗漏因素

  注：在实际应用中，保证核心解释变量与μ不相关即可

标准化回归系数

  • 目的：更为精准的研究影响评价量的重要因素（去除量纲的影响）
  • 方法：
    1. 对数据进行标准化：将原始数据减去它的均数后，再除以该变量的标准差，计算得到新的变量值
    2. 新变量构成的回归方程称为标准化回归方程，回归后相应可得到标准化回归系数
  • 标准化系数的绝对值越大，说明对因变量的影响就越大（只关注显著的回归系数，看p值）

回归系数的解释

对自变量或因变量进行取对数处理

   好处：

  1. 减弱数据的异方差性
  2. 如果变量本身不符合正态分布，取对数后可能渐近服从正态分布
  3. 模型形式的需要，让模型具有经济学意义

   是否需要取对数：

  1. 与市场价值相关的，例如，价格、销售额、工资等都可以取对数
  2. 以年度量的变量，如受教育年限、工作经历等通常不取对数
  3. 比例变量，如失业率、参与率等，两者均可
  4. 变量取值必须是非负数，如果包含0，则可以对y取对数ln(1+y)

四类模型回归系数的解释

  1. 一元线性回归：𝑦 = 𝑎 + 𝑏𝑥 + 𝜇，x每增加1个单位，y平均变化b个单位
  2. 双对数模型：𝑙𝑛𝑦 = 𝑎 + 𝑏𝑙𝑛𝑥 + 𝜇，x每增加1%，y平均变化b%
  3. 半对数模型：𝑦 = 𝑎 + 𝑏𝑙𝑛𝑥 + 𝜇，x每增加1%，y平均变化b/100个单位
  4. 半对数模型：𝑙𝑛𝑦 = 𝑎 + 𝑏𝑥 + 𝜇，x每增加1个单位，y平均变化(100b)%

虚拟变量

  适用场景：自变量中有定性变量，例如性别、地域等

多分类的虚拟变量设置

  为了避免完全多重共线性的影响，引入虚拟变量的个数一般是分类数减1

含有交互项的自变量

  解释变量对一个因变量的偏效应，弹性或半弹性有时取决于另一个解释变量的大小

步骤（Stata）

描述性统计

  对变量类型进行分类

  • 定量数据
  • 定性数据（虚拟变量）

回归

  • 回归系数求解
  • Prob > F （p值）
    作用：是否通过联合显著性检验，原假设为回归系数皆为0
    注：看p值，是否小于0.05
    • 若大于，则在95%置信水平下，无法拒绝原假设，未通过联合显著性检验，模型无意义
    • 若大于，则在95%置信水平下，拒绝原假设，通过联合显著性检验，模型有意义
  • Coef.
    回归系数的估计值
  • Std. Err.
    标准误
  • P>|t|（p值）
    由于回归系数是未知的，对其进行假设检验，原假设为回归系数=0
    注：看p值，是否小于0.05
    • 若大于，则在95%置信水平下，无法拒绝原假设，对应的回归系数为0（与估计值无关）；
    • 若小于，则在95%置信水平下，拒绝原假设，对应的回归系数就是估计值
  • 加入虚拟变量回归
  • 标准化回归 常数项没有标准化回归系数:均值是其本身，经过标准化后变成了0

异方差

  1. 解释：
    对于模型的稳定程度所做的贡献，各个扰动项的方差不相同，那么方差较大的扰动项破坏模型稳定性的程度就较大，或者说它包含的信息量减少
  2. 扰动项存在异方差的影响：
    • OLS估计出来的回归系数是无偏、一致的
    • 假设检验无法使用(标准误（SE）失效，那么构造的t统计量失效了(回归系数的估计值/SE),而p值是根据t统计量计算的),不能看回归系数是否显著
    • OLS估计量不再是最优线性无偏估计量（BLUE）
  3. 解决方法：
    1. 使用OLS + 稳健的标准误
    2. 广义最小二乘估计法GLS
      • 原理：方差较小的数据给予更大的权重，因为方差较小的数据包含的信息较多。
      • 缺点：我们不知道扰动项真实的协方差矩阵，因此只能用样本数据来估计，这样得到的结果不稳健，存在偶然性。

异方差的假设检验

  原假设：扰动项不存在异方差
  P值小于0.05，说明在95%的置信水平下拒绝原假设，即扰动项存在异方差

  1. BP检验
  2. 怀特检验

使用OLS + 稳健的标准误

  • Robust Std. Err.：稳健的标准误

多重共线性

  • VIF:方差膨胀因子
    如果VIF>10，则认为该回归方程存在严重的多重共线性

处理方法

  1. 如果不关心具体的回归系数，而只关心整个方程预测被解释变量的能力，则通常可以 不必理会多重共线性（假设整个方程是显著的）。这是因为，多重共线性的主要后果是使得对单个变量的贡献估计不准，但所有变量的整体效应仍可以较准确地估计
  2. 如果关心具体的回归系数，但多重共线性并不影响所关心变量的显著性，那么也可以不必理会。即使在有方差膨胀的情况下，这些系数依然显著;如果没有多重共线性，则只会更加显著
  3. 如果多重共线性影响到所关心变量的显著性，则需要增大样本容量，剔除导致严重共线性的变量（不要轻易删除，因为可能会有内生性的影响），或对
    模型设定进行修改

逐步回归分析

  1. 向前逐步回归：将自变量逐个引入模型，每引入一个自变量后都要进行检验，显著时才加入回归模型
    缺点：随着以后其他自变量的引入，原来显著的自变量也可能又变为不显著了，但是，并没有将其及时从回归方程中剔除掉
  2. 向后逐步回归：先将所有变量均放入模型，之后尝试将其中一个自变量从模型中剔除，看整个模型解释因变量的变异是否有显著变化，之后将最没有解释力的那个自变量剔除；此过程不断迭代，直到没有自变量符合剔除的条件。
    缺点：一开始把全部变量都引入回归方程，这样计算量比较大（对计算机而言无影响）

   注：

  1. x1 x2 … xk之间不能有完全多重共线性（在每个分类变量中任意去除一个元素即可）
  2. 可以在后面再加参数b和r，即标准化回归系数或稳健标准误

说明

  1. 向前逐步回归和向后逐步回归的结果可能不同
  2. 不要轻易使用逐步回归分析，因为剔除了自变量后很有可能会产生新的问题，例如内生性问题
